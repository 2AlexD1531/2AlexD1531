## 💼 Проекты

🗓️ Explore With Me — Афиша мероприятий (Яндекс Практикум) (завершён)
Микросервисное приложение-афиша, где пользователи могут предлагать мероприятия и собирать компанию для участия в них.
Дипломный проект курса Java-разработчик. 
📦 Архитектура:



| Компонент         | Описание                                            |
| ----------------- | --------------------------------------------------- |
| **Main Service**  | Основной сервис с бизнес-логикой (порт 8080)        |
| **Stats Service** | Сервис статистики просмотров (порт 9090)            |
| **Stats Client**  | HTTP-клиент на RestTemplate для отправки статистики |
| **Database**      | PostgreSQL для хранения данных                      |


✨ Функционал:
Публичный API (без авторизации):

    🔍 Поиск и фильтрация событий (по дате, категории, просмотрам)
    📋 Просмотр подборок событий на главной
    📖 Детальная информация о мероприятиях

Приватный API (для авторизованных пользователей):

    ➕ Создание, редактирование и управление событиями
    📝 Жизненный цикл события: создание → модерация → публикация/отклонение
    🙋‍♂️ Подача заявок на участие и подтверждение участников
    💬 Комментирование мероприятий

Административный API:

    👤 Управление пользователями (CRUD, активация/блокировка)
    🏷️ Управление категориями событий
    📌 Создание и закрепление подборок на главной
    ✅ Модерация событий (публикация или отклонение)

🛠️ Стек: Java 11/21, Spring Boot, Hibernate/JPA, PostgreSQL, Docker, Maven, JUnit, Mockito, REST API 
🚀 Запуск:

mvn clean package
mvn install
docker-compose build
docker-compose up -d

🔗 [github.com/AlexD1533/java-explore-with-me](https://github.com/2AlexD1531/java-explore-with-me)

### 🧠 **Java Quiz App** — Интеллектуальный тестер с ИИ *(активный)*

Полноценное приложение для тестирования знаний Java с генерацией вопросов через ИИ, админ-панелью и полным CI/CD-конвейером.

**🎯 Ключевые фичи:**
- 🤖 **AI-генерация вопросов** через OpenRouter (GPT-3.5/4)
- 🔐 **JWT-авторизация** и Spring Security
- 👨‍💼 **Админ-панель** — первый зарегистрированный пользователь становится админом
- 🐳 **Полная контейнеризация** — бэкенд, фронтенд, БД и Nginx в Docker
- 📊 **Мониторинг** через Gatus с уведомлениями в Telegram
- 🧪 **Покрытие тестами &gt;80%** — JUnit 5 + Mockito + Postman-коллекции

**📦 Архитектура:**
| Компонент | Стек |
|-----------|------|
| Backend | Spring Boot 3.2 + Java 21 + MySQL 8 |
| Frontend | Чистый HTML/CSS/JS (SPA), Nginx |
| AI | OpenRouter API |
| DevOps | Docker + Docker Compose |

🔗 [github.com/2AlexD1531/java-quiz-app](https://github.com/2AlexD1531/java_quiz_app)

### 📱 **SMS Gateway App** — Управление через SMS *(активный)*

Решение для управления цифровыми сервисами через SMS-команды. Работает без интернета — идеально для удалённых зон или аварийных сценариев.

**🚀 Возможности:**
- 🤖 **AI через SMS** — получение ответов от LLM моделей через текстовые сообщения
- 💬 **Telegram-бот** — удалённое управление (просмотр пользователей, чтение сообщений)
- 🌐 **Offline-режим** — доступ к данным без интернет-соединения
- 🖥️ **Лёгкий Web UI** — мониторинг состояния системы
- 📡 **USB-модем** — поддержка Huawei E3372-325 и аналогов

**🛠️ Стек:** Java 21, Spring Boot, Spring Data JPA, MySQL, REST API, Docker

**⚠️ Требования:** USB-модем с настроенной метрикой сети (чтобы трафик не шёл через SIM-карту)

🔗 [github.com/2AlexD1531/sms-gateway](https://github.com/2AlexD1531/SmsGatewayApp)

### 🔄 **ShareIt** — Сервис для шеринга вещей (Яндекс Практикум) *(завершён)*

Микросервисное приложение для совместного использования вещей. Пользователи могут сдавать вещи в аренду, бронировать предметы и оставлять отзывы.

**📦 Архитектура:**
| Компонент | Описание |
|-----------|----------|
| **Gateway** | API-шлюз для валидации и маршрутизации запросов |
| **Server** | Основной сервис с бизнес-логикой |
| **Database** | PostgreSQL для хранения данных |

**✨ Функционал:**
- 👤 **Пользователи** — регистрация, управление профилями
- 📦 **Вещи** — добавление, редактирование, поиск по названию/описанию
- 📅 **Бронирования** — аренда на конкретные даты, подтверждение/отклонение владельцем
- 📝 **Комментарии** — отзывы после завершения аренды
- 📨 **Запросы** — создание запросов на отсутствующие вещи
- 📄 **Пагинация** — постраничный вывод списков

**🛠️ Стек:** Java 21, Spring Boot 3, Hibernate/JPA, PostgreSQL, H2 (тесты), Docker, JUnit 5, Mockito, Maven

🔗 [github.com/2AlexD1531/java-shareit](https://github.com/2AlexD1531/java-shareit)
