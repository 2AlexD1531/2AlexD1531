Полноценное приложение для тестирования знаний Java с генерацией вопросов через ИИ, админ-панелью и полным CI/CD-конвейером.

### 📱 **SMS Gateway App** — Управление через SMS *(активный)*

Решение для управления цифровыми сервисами через SMS-команды. Работает без интернета — идеально для удалённых зон или аварийных сценариев.

**🚀 Возможности:**
- 🤖 **AI через SMS** — получение ответов от LLM моделей через текстовые сообщения
- 💬 **Telegram-бот** — удалённое управление (просмотр пользователей, чтение сообщений)
- 🌐 **Offline-режим** — доступ к данным без интернет-соединения
- 🖥️ **Лёгкий Web UI** — мониторинг состояния системы
- 📡 **USB-модем** — поддержка Huawei E3372-325 и аналогов

**🛠️ Стек:** Java 21, Spring Boot, Spring Data JPA, MySQL, REST API, Docker

**⚠️ Требования:** USB-модем с настроенной метрикой сети (чтобы трафик не шёл через SIM-карту)

🔗 [github.com/2AlexD1531/sms-gateway](https://github.com/2AlexD1531/SmsGatewayApp)
